# Portfolios

`03_portfolios/` is the canonical top governed business container.

## Rules

- Portfolio homes are the first governed business objects beneath the repo root.
- Programs live under portfolio homes.
- Projects live under program homes.
- Task packets live under project homes.
- Each object home contains a primary governing Markdown file and sibling YAML companion.

## Included example

This scaffold includes one example portfolio with one nested program, one project, and one task packet.
