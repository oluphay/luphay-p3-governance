# Portfolios

`03_portfolios/` is the canonical top governed business container.

## Rules

- Portfolio homes are the first governed business objects beneath the repo root.
- Programs live under portfolio homes.
- Projects live under program homes.
- Task packets live under project homes.
- Each object home contains a primary governing Markdown file and sibling YAML companion.

## Included example

This scaffold currently includes one sample/template portfolio hierarchy with
one nested program, one project, and one task packet.

The committed `P3-PORT-0001__human-friendly-title/` lane is a structural
pattern reference only. Do not treat it as a live production portfolio set
unless Alpha explicitly authorises a new governed lane.
