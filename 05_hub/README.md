# Hub

The hub is the controlled intake, staging, and promotion surface for non-canonical
materials.

During the immediate-use operational phase, this area is treated as local staging
only. Durable live work should be created and managed directly under
`03_portfolios/` until the hub promotion path is adopted as a versioned workflow.

## Sub-areas

- `input_hub/` for raw intake and admission processing
- `output_hub/` for staged execution outputs
- `storage_hub/` for stable referenced files retained by durable path
